# Virtual-Research-Match-Maker
A platform to connect researchers with students that allows them to benefit from eachother

First project: Virtual Research Match Making

Two views: 1) Professor/R&D center and 2) student/researcher

Professor view:
Login email/password to be able to update or cancel project
Project title
Subject area 
Abstract/image
Skills necessary
Contact email
Timeline

Student view:
Login with email/password to be able to update profile
Name, Picture, Resume, What kind of research/skills
Subject Browse
Order projects by deadline
Alphabetical order
Search for keywords in projects
